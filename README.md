# IPL Match Intelligence

Ball-by-ball IPL analytics and win prediction project built using Cricsheet data.

## Dataset

- 1,239 IPL matches
- ~295k ball-by-ball records
- Source: Cricsheet

## Current Progress

- Data ingestion pipeline completed
- Match metadata merged successfully
- Exploratory data analysis completed
- Initial cricket trend analysis performed

## Key Insights

- Chasing teams win ~62% of matches
- Modern IPL seasons are significantly higher scoring
- Death overs have highest wicket frequency
- 200+ totals have increased sharply in recent seasons
